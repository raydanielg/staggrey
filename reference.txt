3.6 APPLICATION & ADMISSION MANAGEMENT MODULE (Detailed)
This is a full online application system.
•	Prospective student fills form with: full name, date of birth, phone, email, previous school, program of interest, attachment of certificate (photo upload).
•	Auto-generated application number.
•	Admin can review, comment, accept/reject.
•	If accepted, system auto-sends admission email/SMS with fee payment instructions.
•	After payment of registration fee (tracked), student is automatically enrolled and given student login credentials.
•	All application data exported to Excel.

3.7 STUDENT ACADEMIC MANAGEMENT MODULE (Detailed)
•	Course registration at start of semester (student selects from available courses).
•	Lecturer enters marks (continuous assessment + final exam).
•	System automatically calculates total marks, grade points, GPA, CGPA.
•	Student can view results online and print result slip.
•	Failed courses automatically flagged for retake registration.
•	Transcript request and generation.

3.8 FINANCIAL MANAGEMENT MODULE (FEES)

•	Payments recorded (cash via finance staff or online via mobile money).
•	Automatic receipt generation (PDF + SMS notification).
•	Outstanding balance report.
•	Automatic blocking of exam result viewing if fee balance exceeds threshold (configurable).
